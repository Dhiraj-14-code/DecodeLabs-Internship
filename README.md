# 🚀 Java Projects Collection

This repository contains a set of Java-based projects developed as part of my learning journey in Object-Oriented Programming (OOP) and software development fundamentals.

---

## 📌 Projects Included

### 1. 🎯 Number Guessing Game

* A simple console-based game where the user guesses a randomly generated number.
* Demonstrates:

  * Control flow (loops, conditions)
  * Random number generation
  * User input handling

---

### 2. 🧮 Student Grade Calculator

* Calculates average marks and assigns grades based on input scores.
* Demonstrates:

  * Arrays / input handling
  * Conditional logic
  * Basic data processing

---

### 3. 🏧 ATM Interface (OOP-Based)

* Simulates an ATM system with deposit, withdrawal, and balance checking.
* Key Concepts:

  * Encapsulation
  * Class design (`ATM`, `BankAccount`)
  * Input validation
  * Separation of concerns

---

### 4. 💱 Currency Converter

* Converts amount between currencies using predefined exchange rates.
* Key Concepts:

  * `BigDecimal` for precise financial calculations
  * Input validation with exception handling
  * Scalable conversion using base currency (USD)
